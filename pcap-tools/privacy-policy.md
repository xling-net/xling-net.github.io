# PCAP-Tools Privacy Policy

PCAP-Tools (hereinafter referred to as "the App") is a professional PCAP file processor. We value your privacy and are committed to protecting your data security through a simple and transparent policy.

## 1. App Description and Features

PCAP-Tools is a utility designed for processing PCAP files, offering the following core and auxiliary features:

**Core Features:**
*   **PCAP Splitting**: Supports automatic protocol-based extraction (TCP, UDP, etc.), connection-based session splitting (based on the four-tuple: Source IP/Port, Destination IP/Port), and file cutting by specific block size or time interval.
*   **Efficient File Merging**: Easily merge multiple PCAP/PCAPNG files into a single ordered packet file for unified analysis.

**Auxiliary Features:**
*   **TCPDUMP Command Generator**: Visually configure packet capture filters and automatically generate tcpdump commands.
*   **Wireshark Filter Expression Generator**: Visually configure filter conditions and automatically generate Wireshark filter expressions.

## 2. Privacy Commitments

To protect your privacy, we adhere to the following strict guidelines:

*   **Zero User Data Collection**: We do not collect, store, or transmit any of your personal identifiable information, device information, or usage habit data.
*   **No Registration or Login Required**: The App does not require any account system. You can access all features without registration, ensuring complete isolation between your identity and your data.
*   **Local Processing**: **This is our core privacy statement.** All processing functions provided by the App (including file splitting, merging, and filter configurations) are performed directly on your local device. Your PCAP files are never uploaded to our servers or any third-party cloud services.

## 3. Data Security

Since all operations are executed locally, your data security is protected by your device's operating system (macOS).
